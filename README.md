# FY26 Quality Workshop Voting Page

Static GitHub Pages package for the FY26 Quality Workshop voting detail page.

The page is a front-end helper only: participants review the 10 projects, select exactly 3 items, then open Microsoft Forms for the official submission.

## Publish With GitHub Pages

1. Create a GitHub repository, for example `fy26-qws-voting-pages`.
2. Push this folder to the repository's `main` branch.
3. In the repository, go to `Settings` -> `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select `main` and `/root`, then save.
6. Use the generated Pages URL as the QR code target.
